# You Are What You Prompt: Prompt Quality, Domain Shift, and Uncertainty in Agrifood Vision-Language Models

> 🚧 **This repository is under construction.** Code and documentation will be released upon paper publication.
 
## Overview

Vision-language models enable zero-shot classification through natural language prompts, but performance is sensitive to prompt formulation, especially in specialized domains. This paper evaluates **Zero-shot Prompt Ensembling (ZPE)** in the agrifood domain using CLIP and SigLIP across four datasets and four prompt pools, spanning in-distribution (ID) food and out-of-distribution agricultural benchmarks.

We further introduce **PID (Prompt-based Inconsistency Detection)**, which repurposes prompt disagreement as epistemic uncertainty, improving failure detection under severe domain shift where standard confidence measures collapse.
